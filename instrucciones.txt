========================================================================
GUÍA DE CONFIGURACIÓN Y USO DEL MIGRADOR DE BASES DE DATOS (MYSQL)
========================================================================

Este proyecto está diseñado en Python utilizando Flask para comparar y
migrar cambios estructurales (Tablas, Columnas y Triggers) entre dos
bases de datos MySQL.

REQUISITOS PREVIOS:
1. Python 3.8 o superior instalado en el sistema.
2. Acceso de red y credenciales válidas para ambas bases de datos.

INSTRUCCIONES DE DESPLIEGUE RÁPIDO:

1. Clonar o mover los archivos del proyecto a la carpeta de destino:
   - app.py (Código principal de la aplicación)
   - templates/index.html (Interfaz de usuario)

2. Crear y activar el entorno virtual de Python:
   - Windows:
     python -m venv venv
     venv\Scripts\activate
   - Linux/macOS:
     python3 -m venv venv
     source venv/bin/activate

3. Instalar las dependencias del sistema:
   pip install Flask pymysql

4. Ejecutar la aplicación:
   python app.py

5. Acceder a la interfaz web:
   Abra su navegador e ingrese a http://127.0.0.1:5000

NOTAS DE SEGURIDAD Y OPERACIÓN:
- El sistema realiza un análisis comparativo antes de ejecutar cualquier acción.
- Se recomienda descargar la copia de seguridad (Backup) generada por la
  herramienta antes de aplicar cualquier cambio estructural en el destino.
- El script intenta mitigar conflictos agregando columnas y tablas nuevas,
  pero cambios complejos en restricciones de claves foráneas (Foreign Keys)
  podrían requerir revisión manual según el orden de creación.
